HTML就是超文本标记语言Hypertext Text Marked Language，可以将文本、图片、声音、动画等不同类型的信息整合到一起，在网页上显示出来，还可以从一个文件跳到另一个文件。严格说HTML并不是一门语言，而是一些能够让浏览器看得懂的标记。这是制作一个网页用到的最基础的技术。

万维网和HTML

首先介绍万维网：WWW (World Wide Web）：
百度如是说：是基于客户机/服务器方式的信息发现技术和超文本技术的综合。
万维网与因特网与互联网还是有一些区别的：

  • 万维网：
    是一个系统，由一些资源、标签、传输协议构成的系统。用户通过唯一的标签选择资源，然后通过协议在浏览器上渲染显示。
    打个比方，类似网购过程：资源（文件）就像是商品，是我们用户要买的东西；每个商品都有一个唯一对应的标签牌链接（URL）；决定购买商品时，寻找标签牌链接，然后商品开始快递，从运输到拿到的过程就好比是使用超文本传输协议的过程

  • 因特网（Internet):
    由美国军方研发然后逐渐发展民用，将全世界所有计算机连接起来。概念比万维网更底层一些，通过最基本的一些数据包传递信息，实现连接。因特网包括万维网，也包括其他一些服务，如：电子邮件、IP电话等

  • 互联网（internet）:
    有广义和狭义之分。广义上，能相互之间通信的设备组成的网络就叫互联网；狭义上互联网与因特网含义类似。但注意，也有不是因特网的互联网。

服务器通过超文本标记语言(HTML)把信息组织成为图文并茂的超文本，利用链接从一个站点跳到另个站点。这样一来彻底摆脱了以前查询工具只能按特定路径一步步地查找信息的限制

什么是超文本文件？
超文本文件是指包含有链接关系和多媒体对象的文件。

万维网分为三个基本组成部分：

  • URL（ Universal Resource Locators ）：
    统一资源定位器，相当于网页的一个唯一ID，提供在web上进入目标资源的统一方法和路径

  • HTTP ( Hypertext Transfer Protocol )：
    一种在网络上传输数据的协议，专门用于以超文本或超媒体的形式提供的信息

  • HTML ( Hypertext Text Marked Language ):
    一种文本类、依靠解释的方式执行的超文本标记语言，主要用作编写网页；可以直接用任何文本编辑器打开，查看HTML源码，无需编译直接解释执行

HTML文件的基本结构

<html>
    <head>
        <title>文件标题</title>
    </head>
    <body>
    文件正文
    </body>
</html>

HTML标签

一个HTML文件由许多个标签和元素组成。

  • 标签：用来规定元素属性和位置，小尖括号，可分为，单独标签：相应位置插入元素（如 <br>换行符）和成对标签控制之间的元素（如<title></title>）

  • 元素：通常被一对标签包围；定义：当用一组HTML标签将一段文字包含在中间时，这段文字与包含文字的HTML标签被称为一个元素(从开始标签到结束标签的所有)
    <元素名>要控制的元素</元素名>
    相当于大元素不断嵌套包含小元素，如<html></html>，是最外层的元素，里面又嵌套了很多其他元素

标签之间的关系有：嵌套和并列

一些细节：

  1. 元素名不分大小写；
  2. 标签中写的元素名可以设置一些（任意？）属性值，放在头标签，如<元素名 属性 1=”值1” 属性 2=”值2”>，双引号可省略
  3. 编写时，空格回车在源代码起不了作用，为了便于辨识，一个标签一次换行
  4. 注释：<!-- 注释 --！>

基本结构标签

  • <html> </html>:HTML标签，根标签，页面中最大的标签
  • <head> </head>:头部标签，当中必须包含title标签
  • <title> </title>:标题标签，显示网页的名称，
  • <body> </body>:主体标签，包含页面的所有内容

Web标准

网页开发时，要遵循Web标准，分为三个部分：

  • 结构【身体】：用于对网页元素进行整理和分类，主要是HTML
  • 表现【衣服】：用于设置网页元素的版式、颜色、大小等外观样式，主要是CSS
  • 行为【动作】：用于网页模型的定义以及交互的编写，主要是JavaScript

Web标准提出的最佳体验方案：结构、样式、行为相互分离；即结构写道HTML文件中，表现写到CSS文件中，行为写到JavaScript文件中。
打个比方，结构相当于一个人的身体，没有装饰；而表现就是穿了衣服；行为就是让一动不动的躯体有了各种动作，实现更多功能。
